==========================================================================================

        Este é um simples programa de cadastro de carros construído usando o Django 
              REST Framework. Ele permite que você adicione e visualize 
                     informações sobre carros e seus fabricantes. 
              

==========================================================================================


A aplicação possui dois modelos principais: Carro e Fabricante. 
Você pode acessar a API da aplicação em http://127.0.0.1:8000/api/. 
Aqui estão algumas operações que você pode realizar:

Listar Carros: Acesse http://127.0.0.1:8000/api/carros/ para 
listar todos os carros cadastrados.

Criar Carro: Use um cliente API (como Postman) para enviar um 
POST request para http://127.0.0.1:8000/api/carros/ com os dados 
do carro que deseja adicionar.

Listar Fabricantes: Acesse http://127.0.0.1:8000/api/fabricantes/ 
para listar todos os fabricantes cadastrados.

Criar Fabricante: Use um cliente API (como Postman) para enviar um 
POST request para http://127.0.0.1:8000/api/fabricantes/ com os dados 
do fabricante que deseja adicionar.

Espero que este README tenha ajudado a entender como usar 
a aplicação de cadastro de carros.
